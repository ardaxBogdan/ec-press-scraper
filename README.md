# EC Press Scraper (Scrapy)

Scraper tiskových zpráv Evropské komise (Press corner) postavený na Scrapy.

- jazyk: Python 3.10+
- framework: Scrapy
- perzistence: JSON Lines (`data/press_releases.jsonl`)

## Instalace

```bash
git clone <tento-repozitar>
cd ec-press-scraper
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
# ec-press-scraper
